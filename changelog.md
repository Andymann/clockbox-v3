This is the changelog of the clockbox v3. Official version started at version 3.47 so don't worry to find little to no information prior to this.

## v3.54
Fix bug: Fading from 130 to 90 BPM left the clock being stuck at 91 BPM.

## v3.53
Fix bug: Double-clicking TAP button set tempo to 240 BPM.

## v3.52
Fix bug: Preset buttons did not work on first press.

## v3.51
Add brightness settings for LEDs.

## v3.50
Make LED 5 blink on incoming MIDI data.

## v3.49
Long click on preset-button fades into new tempo, short click immediately changes tempo.

## v3.48
November 2025  
  
Add cv/gate functionality when in follow-mode. Mode 'QRS Stop-Start' is now the default mode on first start.

## v3.47
xxx 2025  
  
Most things actually work =). This version proved to be stable and reliable for ~6 months in beta-testing.
